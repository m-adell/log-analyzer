# Log Analyzer 
A Bash script that analyzes Apache web server logs and generates a structured daile report.
# Features
- Top 10 IPs accessing the server
- Top 10 most requested URLs
- Response code summary (200, 403, 404, 500)
- Saves report with timestamp
# Thecnologies
- Bash
- awk, grep, sort, uniq
- cron for schedule analyzes
